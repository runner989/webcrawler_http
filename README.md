# Web Crawler

## Description

A web crawler that fetches and reports internal and external links.

## Installation

```
bash
git clone https://github.com/runner989/webccrawler_http.git
cd webcrawler_http
npm install
```

## Running

```
node run main.js https://site-to-crawl.com
```
